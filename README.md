\#\#\# Usage Notes

**Seariously** is an interactive generative artwork inspired by Annie Leonard’s quote, *“There is no such thing as ‘away.’ When we throw anything away it must go somewhere,”* and the environmental issues addressed in Sustainable Development Goal 14: Life Below Water. The project highlights how waste discarded on land can eventually reach the ocean, threatening marine ecosystems and the wildlife that depend on them. Through procedural generation, each visit presents a unique underwater composition featuring randomly arranged marine creatures alongside their waste-covered counterparts, encouraging viewers to reflect on the growing impact of marine pollution and ask, *"How much of the ocean is still natural?"*

Users can explore the artwork by scrolling through the webpage to experience the layered underwater environment. As the project incorporates generative coding, every page reload creates a unique composition with different marine creatures, waste-covered creatures, and animated environmental elements such as bubbles, sparkles, and drifting movement. To generate a new composition, simply refresh the webpage.

\#\#\# Installation Instructions

Extract the contents of COMM2754-2026-S2-A1w04-Seariously-project-files.zip. Open the project in Visual Studio Code and run `index.html` using the Live Server extension.
